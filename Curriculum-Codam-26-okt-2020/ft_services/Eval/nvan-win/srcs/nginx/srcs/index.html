kubectl Cheat Sheet

kubectl get nodes -> listing of all nodes in a cluster
kubectl describe node <node-name> -> detailed information about node
kubectl get pods -> listing of all pods in a cluster
kubectl get deployments -> listing of all deployments in a cluster
kubectl get services -> listing of all services in a cluster
kubectl scale deployment nginx --replicas=0 -n default -> Scale a replicaset
